Detailed Description

template<int dim, typename PropertiesIndex>
class ParticleWallContactForce< dim, PropertiesIndex >

Base interface for classes that carry out the calculation of particle-wall contact force.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.

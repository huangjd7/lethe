Detailed Description

template<int dim, typename PropertiesIndex>
class ParticleWallDMTForce< dim, PropertiesIndex >

Calculation of the DMT particle-wall contact force using the information obtained from the fine search and physical properties of particles and walls.

Template Parameters
dimAn integer that denotes the number of spatial dimensions.
PropertiesIndexIndex of the properties used within the ParticleHandler.

◆ initialize_particle_wall_properties()

template<int dim, typename PropertiesIndex >
void ParticleWallDMTForce< dim, PropertiesIndex >::initialize_particle_wall_properties ( const DEMSolverParameters< dim > &  dem_parameters)
inlineprivate

Initialize every particle-wall properties required for the force calculation. This function is required for the DMT force model, since the dmt_cut_off_distance attribute needs the be set to const. To do so, the effective surface energy and of the effective hamaker contant need to be initialized before hand. For code clarity, we also initialized every other parameter, otherwise the initialization would take place at two different places in the code.
